Arborfield Local History

Repository initialized for GitHub Pages. This is a placeholder page.

If you want me to push your website files from OneDrive into this repository, either upload a zip of your site here or follow the instructions in the README from your local machine.

Quick commands to push from your OneDrive folder

# From your OneDrive site folder (not a git repo)
cd "/path/to/OneDrive/YourSiteFolder"

git init
git add .
git commit -m "Initial commit: website from OneDrive"
git branch -M main
git remote add origin https://github.com/Patricia-Fryer/arborfieldlocalhistory.git
git push -u origin main

After you push, enable Pages in Settings → Pages and choose the main branch and the /docs folder as the publishing source.

If you'd like me to set the Pages publishing source for you or push files on your behalf, tell me and provide the site files (zip) or confirm I have permission to change repository settings.
